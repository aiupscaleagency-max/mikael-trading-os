Koppla dina konton

Du betalar bara för software + agenterna. Trading sker på ditt eget Binance-konto med dina pengar. AI-anrop går via din egen Anthropic-nyckel.

🧠

Anthropic API-nyckel KRÄVS

Varför: Hanna, Albert och alla 8 specialister är drivna av Claude. Du betalar Anthropic direkt för deras tokens — vi tar 0% av AI-kostnaderna.

Skapa nyckel: console.anthropic.com/settings/keys → Create Key → kopiera (börjar med sk-ant-api03-...)
💡 Tips: börja med ~$10 i krediter på Anthropic-kontot. Det räcker för 30+ dagars körning.
🟡

Binance API KRÄVS

Varför: Hanna lägger ordrar direkt på ditt Binance-konto. Vi rör aldrig dina pengar.

Skapa nyckel: binance.com → API Management → Create API → ✅ Enable Spot Trading → IP-whitelist din server-IP

Säkerhet: aktivera ALDRIG "Enable Withdrawals". Om en attackerare får din nyckel kan de bara handla, inte ta ut pengar.
💡 Använd Test-läge först — då går allt mot Binance testnet utan riktiga pengar.
🔍

Perplexity (Lars) VALFRI

Varför: Lars hämtar färska nyheter, makro-händelser och geopolitik via Perplexity Sonar innan teamet analyserar. ~$0.005/anrop = ~$0.07/vecka.

Skapa nyckel: perplexity.ai/settings/api

Hoppa över om du inte vill använda Lars — teamet kör ändå utan färsk web-research.
💱

Oanda (Forex) VALFRI

Varför: Om du vill att Hanna handlar valutor (EUR/USD, GBP/USD, USD/JPY...). Endast crypto utan Oanda — ingen forex.

Skapa konto: oanda.com → Practice (demo) → Manage API Access → Generate API Token

Hoppa över om du bara vill handla crypto.

🔒 Nycklar lagras krypterat i din browser (AES-256). I Fas 2 → krypteras i Supabase med row-level security.
Du kan ändra dem senare från Settings.
